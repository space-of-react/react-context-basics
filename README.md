# Контекст в React для управления состоянием

Набор React-заданий по работе с контекстом: каждое задание и его решение запускаются отдельной npm-командой.

## Установка

```bash
git clone https://github.com/your-username/react-context-basics.git
cd react-context-basics
npm install
```

## Как запускать задания

Запуск **задачи** (например, задание 01):

```bash
npm run task:01
```

Запуск **решения**:

```bash
npm run solution-01
```

Дальше открой адрес, который выведет Vite в терминале (обычно `http://localhost:5173`).

## Как проходить курс

Курс разделён на упражнения. Каждое упражнение содержит файлы `*.проблема.jsx` и `*.решение.jsx`.

### Чтобы пройти упражнение:

1. **Запусти задание:**
   ```bash
   npm run task:01
   ```

2. **Попробуй решить самостоятельно:**
   - Открой файл задания в `src/` (например, `src/01-введение-в-контекст.проблема.jsx`)
   - Изучи код и выполни условие из подзаголовка на странице
   - Используй [документацию React](https://react.dev/) при необходимости
   - Проверь результат в браузере (`http://localhost:5173`)

3. **Проверь решение:**
   
   Если справился или застрял — запусти готовое решение:
   ```bash
   npm run solution-01
   ```
   
   Сравни своё решение с предложенным. Возможно, твоё решение лучше!

## Советы по обучению

- **Не спеши смотреть решение** — попробуй найти подход самостоятельно
- **Используй документацию** — [React Docs](https://react.dev/) и раздел про [Context](https://react.dev/learn/passing-data-deeply-with-context) помогут
- **Экспериментируй** — меняй код и смотри, что происходит
- **Сравнивай решения** — твой подход может быть интереснее готового
